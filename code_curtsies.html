curtsies

Curtsies, built to be the front-end of bpython, is an open-source library with various terminal visual operations: cursor movement, input detection, and colored strings. It's found all over GitHub helping to make cool command line tools. Along with bypthon, this summer, I was contracted to contribute to and improve upon curtsies.

My contributions:

  • Auto formatted library, established clear formatting guidelines, and implemented git ignore to preserve git blame history
  • Programmed a command line clone of tron light cycles as an example for the features of curtsies (seen above!)
  • Used my familiarity with the code to fix close some small issues open on GitHub
  • Improved developer relations by adding developer introduction and improving ReadTheDocs, lowering the barriers for future contributors
