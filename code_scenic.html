scenify

For my senior capstone project, I built a web app that generates optimally scenic walking routes from point A to point B using AI.

The tool uses a Convolutional Neural Network trained on the MIT places 205 dataset to classify geotagged Flickr images in an area as "natural scenic" or "urban scenic", allowing users to view the type of scenery they'd like in their desired location.

This project was super fun, and I got to learn about a lot of things: geospatial data, neural networks, APIs, traversal algorithms, databases, as well as possible answers to the question: how do we quantify scenicness?.
