As part of a team created and sponsored by Thomas Ballinger, I spent summer 2020 consulting for the some open source python libraries. (Blog post coming soon!)

bpython

bython is a fancy interface to the python interpreter used by thousands of programmers of all experience levels. Features include autocompletion, auto suggesting, syntax highlighting, auto-indentation, the ability to edit your session in an editor, and my personal favorite: undo.

bpython/curtsies
