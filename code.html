Software Consulting

As part of a team created and sponsored by Thomas Ballinger, I spent summer 2020 consulting for the following open source python libraries. (Blog post coming soon!)

bpython

bython is a fancy interface to the python interpreter used by thousands of programmers of all experience levels. I swear by it. Features include autocomplete, quick attribute inspection of objects, syntax highlighting, auto-indentation, the ability to edit your session in an external editor, and my personal favorite: undo.

bpython/curtsies

Curtsies, built to be the front-end of bpython, is a library with various terminal visual operations, like cursor movement, input detection, and colored strings. It allows for cool command line tools which don't require lots of scrolling to show images. I expect to use curtsies a lot for my school projects. Here's a game I made to demonstrate.
