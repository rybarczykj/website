Software Consulting

As part of a team created and sponsored by Thomas Ballinger, I spent summer 2020 consulting for some open-source software.

bpython

bython is a fancy interface to the python interpreter used by tens of thousands of developers wordwide. Features include attribute completion, in-line syntax highlighting, auto-indentation, expected parameter lists, and my personal favorite: undo. Check out this in-browser demo I made to demonstrate.

bpython/curtsies

Curtsies, built to be the front-end of bpython, is a library with various terminal visual operations: cursor movement, input detection, and colored strings. It's found all over GitHub helping to make cool command line tools, like this game I made.
